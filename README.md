# Computer-Technology-Vocabulary

开发思路：
1. 在 wordlist.dat 中添加单词列表
2. 调用 API  获取单词音标、定义、翻译、备注等信息
3. 填充到 README.md 中，table 排序

自动更新：
```build xx ==> php index.php && git commit -am "commit info" && git push```
